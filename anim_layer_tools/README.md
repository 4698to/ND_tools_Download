#  动画层工具


![](https://gitee.com/to4698/ND_tools/raw/master/img/20190819165627.png)

一起5个功能

选择对象: 选择有动画层对象。

塌陷动画: 免去运动面板操作的快速塌陷

向下塌陷: 以动画层的帧数为准塌陷动画层。塌陷之后不是满帧。

向上塌陷: 以基础层的帧数为准塌陷动画层。

帧对齐: 选中多个物体,自动以第一个选中物体帧数为准,自动删除多余帧
